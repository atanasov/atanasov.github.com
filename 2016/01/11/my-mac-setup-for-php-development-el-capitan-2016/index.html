My Mac Setup for PHP development - El Capitan 2016

Atanas Atanasov's Blog
2016-01-11

Install Homebew

ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
sudo chown -R $(whoami) /usr/local
brew prune
echo 'export PATH="/usr/local/sbin:$PATH"' &gt;&gt; ~/.bash_profile
brew update

Install some apps for Mac with cask

brew install caskroom/cask/brew-cask
brew cask install google-chrome
brew cask install skype
brew cask install lastpass
brew cask install macdown
brew cask install sourcetree
brew cask install flash-player

Install QuickView plugins

brew cask install qlcolorcode
brew cask install qlstephen
brew cask install qlmarkdown
brew cask install quicklook-json
brew cask install qlprettypatch
brew cask install quicklook-csv
brew cask install betterzipql
brew cask install webpquicklook
brew cask install suspicious-package

Install PHP

brew tap homebrew/dupes
brew tap homebrew/versions
brew tap homebrew/homebrew-php
brew install php56 --with-homebrew-curl --without-bz2 --without-apache --without-fpm --without-ldap --without-mysql
brew install mcrypt php56-mcrypt

Install Composer

curl -sS https://getcomposer.org/installer | php
sudo mv composer.phar /usr/local/bin/composer

Install zsh

brew install zsh zsh-completions
curl -L https://github.com/robbyrussell/oh-my-zsh/raw/master/tools/install.sh | sh
chsh -s /bin/zsh

Install Sublime Text and link it

brew cask install sublime-text
ln -s /Applications/Sublime\ Text.app/Contents/SharedSupport/bin/subl /usr/local/bin/subl

Install Vagrant & VirtualBox & Homestead

brew cask install vagrant
brew cask install virtualbox

go to your home folder and run

vagrant box add laravel/homestead
mkdir Code
cd Code
git clone https://github.com/laravel/homestead Homestead

go to Homestead folder and Init homestead

cd Homestead
bash init.sh

Configure site mapping subl ~/.homestead/Homestead.yaml add to .bash_profile or .zshrc - for easy access to vagrant up

alias homestead='function __homestead() { (cd ~/Homestead &amp;&amp; vagrant $*); unset -f __homestead; }; __homestead'

when changing homestead.yaml file do provision to update settings in vagrant: homestead provision start server with: homestead up stop server with: homestead halt go with ssh to virtual machine homestead ssh

Install PHPMyAdmin in Virtual machine

SSH into your Homestead box and cd to your code/projects directory run this great autoinstall script

curl -sS https://raw.githubusercontent.com/grrnikos/pma/master/pma.sh | sh

Open the /etc/hosts file on your main machine and add 192.168.10.10 phpmyadmin.app
